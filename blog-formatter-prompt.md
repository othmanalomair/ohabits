# Prompt لتنسيق المدونات

## التعليمات

أبي منك ترتب لي هالمدونة بصيغة Markdown مع الالتزام بالقواعد التالية:

١. **لا تغير كلامي أو أسلوبي أو اللهجة** — خلها كويتية مثل ما هي
٢. رتب الأفكار بشكل منطقي مع عناوين واضحة
٣. استخدم **bold** للنقاط المهمة
٤. استخدم `---` للفصل بين الأقسام الرئيسية
٥. إذا فيه قوائم أو نقاط مرقمة، خلها مرتبة
٦. **لا تضيف أي كلام من عندك** أو تحليلات أو نصايح
٧. عطني النص المنسق فقط، جاهز للنسخ
